# Projeto 03 — SQL — Analise de Dados

> Modele um banco relacional e escreva queries analiticas avancadas

## Contexto

SQL e a linguagem universal dos dados. Neste projeto, voce vai criar um banco relacional com modelagem dimensional (star schema), popular com dados ficticios e escrever queries que respondem perguntas reais de negocio.

## Requisitos

1. Criar um banco SQLite com pelo menos 4 tabelas relacionadas
2. Implementar modelagem star schema (fato + dimensoes)
3. Popular com dados ficticios realistas (minimo 500 registros)
4. Escrever pelo menos 6 queries analiticas usando: JOINs, GROUP BY, CTEs, Window Functions, Subqueries, Agregacoes
5. Cada query deve responder uma pergunta de negocio
6. Documentar o schema e as queries

## Criterios de Pronto

- [ ] Banco SQLite com schema bem modelado
- [ ] 6+ queries analiticas funcionando
- [ ] Cada query responde uma pergunta de negocio clara
- [ ] Schema documentado com diagrama

## Skills Praticadas

SQL, SQLite, Modelagem Dimensional, CTEs, Window Functions, JOINs

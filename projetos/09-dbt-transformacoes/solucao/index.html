Conceitos dbt

Seeds

CSVs carregados no banco com dbt seed. Dados de referencia.

Staging

Views que limpam dados brutos. Prefixo stg_. Materialized: view.

Marts

Tabelas finais pra analise. dim_ e fct_. Materialized: table.

ref()

Referencia entre models. dbt resolve dependencias automaticamente.

Tests

unique, not_null, accepted_values, relationships. Garante qualidade.

Docs

Descricoes em schema.yml. dbt docs generate cria site completo.

# Projeto 10 — GCP — BigQuery Fundamentals

> Ingira dados no BigQuery e otimize queries na nuvem

## Contexto

Primeiro projeto na nuvem! Voce vai usar o Google Cloud Platform para fazer upload de dados, carrega-los no BigQuery, otimizar queries e entender custos. Tudo usando o free tier do GCP.

## Requisitos

1. Criar uma conta GCP (free tier)
2. Criar um bucket no Cloud Storage e fazer upload de dados
3. Carregar dados do GCS no BigQuery
4. Criar tabelas com particionamento e clustering
5. Escrever queries otimizadas
6. Configurar IAM basico (service account)
7. Analisar custos de processamento

## Criterios de Pronto

- [ ] Dataset no BigQuery com dados carregados
- [ ] Tabelas particionadas e clusterizadas
- [ ] Queries otimizadas com analise de custos
- [ ] IAM configurado com least privilege

## Skills Praticadas

BigQuery, Cloud Storage, IAM, particionamento, clustering, custos GCP

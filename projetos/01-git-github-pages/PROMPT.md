# Prompt para Portfolio — Projeto 01: Git + GitHub Pages

Abra o [Claude](https://claude.ai) (gratuito) e cole o prompt abaixo.

---

## Prompt

```
Acesse o código de referência deste portfolio de Data Engineer:
https://github.com/guicolla/roadmap-data-engineering/tree/main/projetos/01-git-github-pages/solucao

Com base nesse exemplo, gere um index.html para MEU portfolio pessoal de Data Engineering.

MINHAS INFORMAÇÕES (preencha antes de enviar):
- Nome: [SEU NOME]
- Título: Aspirante a Data Engineer (ou seu título atual)
- Sobre mim: [2-3 frases sobre você, seu background e objetivo]
- Skills: [liste suas skills - ex: Python, SQL, Git, Linux]
- LinkedIn: [seu link]
- GitHub: [seu link]
- Email: [seu email]

REQUISITOS:
- HTML único, responsivo, pronto para GitHub Pages
- Tema escuro com estilo moderno e profissional
- Seção "Sobre Mim"
- Seção "Skills" com badges visuais
- Seção "Projetos" com 12 cards (um para cada projeto do roadmap)
  - Projetos ainda não feitos devem aparecer como "Em breve" ou com visual de bloqueado
  - Cada card deve ter espaço para: título, descrição, tecnologias e link para a GitHub Pages do projeto
- Seção "Contato" com links para LinkedIn, GitHub e email
- Footer com créditos
- Animações suaves (scroll reveal, hover effects)
- O estilo NÃO deve ser GC Data (laranja/escuro) — use cores e estilo que combinem comigo

O HTML deve funcionar como arquivo único sem dependências externas (apenas Google Fonts via CDN é permitido).
```

---

## Como usar

1. Abra [claude.ai](https://claude.ai) (conta gratuita)
2. Cole o prompt acima
3. **Preencha suas informações** nos campos entre colchetes
4. Envie e aguarde o Claude gerar o HTML
5. Copie o código e salve como `index.html` no seu repositório
6. Faça commit e push
7. Ative GitHub Pages (Settings → Pages → main branch → Save)
8. Acesse `https://seu-usuario.github.io/seu-repo`

## Dica

Depois de gerar a primeira versão, você pode pedir ajustes ao Claude:
- "Mude as cores para azul e cinza escuro"
- "Adicione um efeito de digitação no título"
- "Coloque ícones nos cards de skills"

Quanto mais você personalizar, mais único fica seu portfolio.

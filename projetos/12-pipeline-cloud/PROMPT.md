# Prompt para Portfolio — Projeto 12: Pipeline Cloud Completo

Cole o prompt abaixo no ChatGPT, Claude ou outra IA para gerar a pagina HTML do seu portfolio para este projeto.

---

## Prompt

```
Gere um HTML responsivo e interativo para meu portfolio de Data Engineering.

PROJETO: Pipeline Cloud Completo
DESCRICAO: Pipeline de producao: GCS + Composer + dbt + BigQuery + Terraform
TECNOLOGIAS: GCP, Cloud Composer, dbt, BigQuery, Terraform, CI/CD, GitHub Actions

CONTEXTO DO PORTFOLIO:
- diagrama de arquitetura cloud interativo com componentes GCP clicaveis, fluxo de dados animado e detalhes de cada servico

REQUISITOS TECNICOS:
- HTML, CSS, JavaScript, SVG ou Canvas
- Responsivo (mobile-first)
- Tema escuro com fundo #111111
- Cor accent: #F5900D (laranja)
- Font body: Space Grotesk (Google Fonts)
- Font code: JetBrains Mono (Google Fonts)
- Deve funcionar como arquivo HTML unico (sem dependencias externas alem de CDNs)
- Incluir link de volta para o portfolio principal

SECOES OBRIGATORIAS:
1. Header com titulo do projeto e badges das tecnologias
2. Sobre o Projeto (descricao e objetivo)
3. Arquitetura / Fluxo (diagrama visual ou interativo)
4. Codigo em destaque (com syntax highlighting)
5. Resultados / Demo interativa
6. Aprendizados (o que voce aprendeu)
7. Footer com link para GitHub e portfolio

Gere o HTML completo, pronto para publicar no GitHub Pages.
```

---

## Como usar

1. Copie o prompt acima
2. Cole no ChatGPT, Claude ou outra IA
3. Personalize com detalhes do SEU projeto (dados que usou, queries que escreveu, etc.)
4. Salve como `index.html` no repositorio do projeto
5. Ative GitHub Pages nas configuracoes do repo
6. Adicione o link no seu portfolio principal

> 💡 Dica: quanto mais detalhes voce adicionar sobre o que VOCE fez, melhor sera a pagina gerada.

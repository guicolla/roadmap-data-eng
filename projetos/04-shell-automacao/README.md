# Projeto 04 — Shell — Automacao

> Automatize tarefas repetitivas com scripts Bash

## Contexto

Automacao e uma skill essencial. Neste projeto, voce vai escrever scripts Bash que automatizam tarefas do dia a dia de um Data Engineer: setup de ambiente, backup de dados, processamento batch e agendamento com cron.

## Requisitos

1. Escrever um script de setup de ambiente (instalar dependencias, configurar variaveis)
2. Escrever um script de backup automatico de arquivos/banco
3. Escrever um script de processamento batch (ex: renomear, mover, comprimir arquivos)
4. Configurar agendamento com cron
5. Usar variaveis, condicionais, loops e funcoes
6. Tratar erros e gerar logs

## Criterios de Pronto

- [ ] 3+ scripts funcionais e documentados
- [ ] Crontab configurado
- [ ] Tratamento de erros e logs implementados

## Skills Praticadas

Bash, Shell Scripting, cron, variaveis de ambiente, logs

# Projeto 05 — Linux — Ambiente de Dados

> Domine o terminal e configure um ambiente profissional de Data Engineer

## Contexto

Linux e o sistema operacional padrao em ambientes de dados. Neste projeto, voce vai documentar a configuracao completa de um ambiente de Data Engineer: gerenciamento de pacotes, permissoes, processos, SSH e monitoramento.

## Requisitos

1. Documentar instalacao e configuracao de um ambiente Linux
2. Gerenciamento de pacotes (apt/yum)
3. Permissoes de arquivos e diretorios (chmod, chown)
4. Gerenciamento de processos (ps, top, kill, nohup)
5. Configuracao de SSH (chaves, config)
6. Variaveis de ambiente e .bashrc/.zshrc
7. Monitoramento basico (df, du, free, htop)

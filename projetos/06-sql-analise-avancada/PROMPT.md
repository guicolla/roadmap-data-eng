## Prompt

```
Acesse o código de referência deste projeto de SQL avançado com dados do Spotify:
https://github.com/guicolla/roadmap-data-eng/tree/main/projetos/06-sql-analise-avancada/solucao

Com base nesse exemplo, gere um index.html interativo para meu portfolio de Data Engineering.

O HTML deve ser um SQL Explorer avançado com:

1. Header com título e badges (SQL, CTEs, Window Functions, Star Schema)
2. Diagrama visual do star schema (tabela fato streams no centro, dimensões ao redor)
3. Sidebar com 6 queries clicáveis, cada uma mostrando o conceito SQL que usa
4. Área principal com:
   - Pergunta de negócio
   - Explicação do conceito SQL usado (CTE, Window Function, etc) com visual didático
   - Código SQL com syntax highlighting
   - Botão "Executar" que revela resultados em tabela
5. Seção "Conceitos Aprendidos" com cards visuais (CTE, RANK, LAG, etc)
6. Footer

REQUISITOS TÉCNICOS:
- HTML único, responsivo, pronto para GitHub Pages
- Tema escuro com verde Spotify (#1DB954) como accent
- Diagrama do star schema visual e claro
- Syntax highlighting no SQL
- Funcionar sem dependências externas (apenas Google Fonts via CDN)
```
